# Product Description (Landing Copy Draft)

## Hero
Production Chat Prop je web-first alat za kreiranje, playback i export simuliranih chat razgovora za film, reklame i digitalnu produkciju.

## Problem
Većina postojećih “fake chat” alata nije prilagođena produkcijskom radu: playback je ograničen, export je slab, a vizuali često ulaze u pravno sive zone.

## Rješenje
Production Chat Prop donosi production-safe workflow:
- uređivanje scena, likova i poruka bez koda
- kontrolirani playback po timestampovima (play/pause/restart/scrub)
- screenshot export i video fallback workflow za post-produkciju
- originalni, brand-safe chat stilovi

## Za koga je
- produkcijske kuće i reklamne agencije
- art odjeli, rekviziteri i asistenti režije
- video editori, motion dizajneri i content timovi

## Core vrijednosti
- brzina pripreme scena
- predvidljivost playbacka na setu
- konzistentan output za storyboard, pitch i finalni edit

## Primarni CTA
- "Pokreni demo projekt"
- "Isprobaj playback flow"
- "Izvezi screenshot u jednom kliku"

## Kratki product pitch
Production Chat Prop pomaže timovima da u nekoliko minuta pripreme uvjerljive chat sekvence za set i montažu, uz jednostavan editor i kontrolirani playback bez oslanjanja na copycat UI rješenja.
